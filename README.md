# Simple Calculator

A lightweight and intuitive **Simple Calculator** application built to perform fundamental arithmetic operations quickly and accurately. This project serves as an easy-to-use utility tool and a clean demonstration of foundational programming logic, error handling, and user interface design.

---

## 🚀 Features

* **Basic Arithmetic:** Supports addition, subtraction, multiplication, and division.
* **Input Validation:** Prevents common application crashes, such as handling division by zero errors.
* **Clear Functionality:** Reset or clear inputs instantly for consecutive calculations.
* **Responsive Layout:** Optimized interface that scales smoothly across mobile, tablet, and desktop screens.

---

## 🛠️ Tech Stack

* **Language: JavaScript
* **Framework/Library: Vanilla HTML & CSS
* **Styling:CSS3
